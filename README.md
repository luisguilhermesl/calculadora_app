## 📖 Sobre o Projeto

Este é um aplicativo de **Calculadora** simples e funcional desenvolvido totalmente em **Java**. A interface gráfica do usuário (GUI) foi construída utilizando a robusta biblioteca **Swing**. É um excelente projeto que demonstra a aplicação prática de conceitos como programação orientada a objetos, manipulação de eventos e design de interfaces para desktop em Java.

## ✨ Funcionalidades

- **Operações Matemáticas Básicas:** Suporte nativo para Adição (+), Subtração (-), Multiplicação (*) e Divisão (/).
- **Suporte a Decimais:** Cálculos precisos utilizando números de ponto flutuante (botão `.`).
- **Interação pelo Teclado:** A calculadora possui "KeyListeners" que permitem usar o teclado físico (Teclado Numérico) para inserir números e operadores, além das teclas `Enter` para exibir o resultado e `Backspace` para apagar.
- **Limpeza e Correção:** - Botão `C` para limpar todo o visor e reiniciar o cálculo.
  - Botão `<` para apagar apenas o último dígito inserido (útil para pequenos erros).

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** Java (JDK 8 ou superior)
- **Interface Gráfica:** Java Swing / AWT
- **IDE Recomendada:** Apache NetBeans
- **Build Tool:** Apache Ant

## 🚀 Como Executar o Projeto

Siga os passos abaixo para compilar e rodar o projeto na sua máquina local:

### Pré-requisitos
Certifique-se de ter o [Java Development Kit (JDK)](https://www.oracle.com/java/technologies/downloads/) instalado na sua máquina. O uso do [NetBeans IDE](https://netbeans.apache.org/) é altamente recomendado, pois o projeto foi gerado com ele.

### Passo a Passo

1. **Clone o repositório:**
 
   git clone [https://github.com/luisguilhermesl/calculadora_app.git](https://github.com/luisguilhermesl/calculadora_app.git)
 

2. **Abra o projeto na IDE:**
- Inicie o NetBeans IDE.
- Vá em File > Open Project... (Arquivo > Abrir Projeto).
- Navegue até a pasta onde você clonou o projeto e selecione a pasta calculadora_app.

3. **Execute a aplicação:**
- Na aba "Projects" do NetBeans, clique com o botão direito no projeto calculadora_app.
- Selecione Run (Executar) ou simplesmente pressione F6.

(Alternativa via linha de comando com Apache Ant):
Se você tiver o Ant configurado nas variáveis de ambiente do seu sistema, navegue até o diretório do projeto no terminal e execute:

```bash
ant run
 ```
